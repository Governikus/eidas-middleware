To use the online id function you need

1. An identity card / electronic residence permit

(issued from 01.11.2010 or 01.09.2011) with unlocked online ID function and your personal PIN

2. A card reader / Android device for direct readout via NFC

An overview of suitable card readers or mobile devices can be found at ausweisapp.bund.de/en/ (card reader / smartphones / tablets)

3. The AusweisApp2

The App can be downloaded for free here:
